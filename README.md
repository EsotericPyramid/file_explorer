# File Explorer

a basic file explorer because Finder sucks

## Controls

- WASD and arrow keys: move in the obvious directions
- R: move file explorer root to its parent
- F: move file explorer root the child containing the cursor
- E: open and close directories
- Q: exit (and output the cursor's path to stdout)
- V: update the file tree
- C: read the cursor's file (uses less)

## Misc

- you can use ```cd `file_explorer` ``` to cd to a selected directory
